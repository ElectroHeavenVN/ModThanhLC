3. Về quản lý tài khoản [QLTK]
3.1. Cách kích hoạt QLTK
Bước 1: Mở [ThanhLc] Product License.exe, copy Mã kích hoạt.
Bước 2: Truy cập phần Lịch sử mua tool trên web thanhlc.com, ấn EDIT Mod tương ứng.
Bước 3: Nhập mã kích hoạt đã copy bên trên vào ô EDIT, rồi ấn Lưu. 
Bước 4: Mở QLTK.

3.2. Cách chuyển file dataAccount từ phiên bản cũ sang phiên bản mới
Để tránh việc phải nhập lại tài khoản nhiều lần, vào folder của phiên bản cũ, kiếm folder tên Data, tìm folder tên QLTK, copy file dataAccount.json nằm trong folder QLTK. Sau đó dán (paste) sang folder Data/QLTK của phiên bản mới. Sau đó mở QLTK lên để load lại data.
